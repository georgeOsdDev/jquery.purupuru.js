# jquery.purupur.js
Add purupuru animation easily..

## Usage

	<script src="jquery.min.js"></script>
	<script src="jquery.purupuru.js"></script>
	<script type="text/javascript">
			$(element).purupuru();
	</script>

>See also > [Demo](http://georgeosddev.github.io/jquery.purupuru.js/)
