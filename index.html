jquery.purupuru.js

add purupuru animation easily.

This plugin have been tested and approved for use only in the webkit-browser.
If there is a bug, enjoy hacking.

Usage

  $(element).purupuru();

Options

  • direction: Integer(0:horizontal,1:vertical,2:multi) : default 0
  • infinite: Boolean : default false
  • animationDelay: Integer (millisec) : default 0
  • animationDuration: Integer (millisec) : default 250
  • purupuruBuffer: Integer (px) : default 10

Demo

click me!

  $("#purupuruBox0").on("click",function(){
    $(this).purupuru();
  });

click me!

  $("#purupuruBox1").on("click",function(){
    $(this).purupuru({direction:1});
  });
